Dados

O que são dados perdidos?

  • Dados perdidos são dados não coletados de um ou mais participantes, para uma ou mais variáveis (Douglas G. Altman and Bland 2007).


Qual o problema de um estudo ter dados perdidos?

  • Uma grande quantidade de dados perdidos pode comprometer a integridade científica do estudo, considerando-se que o tamanho da amostra foi estimado para observar um determinado tamanho de efeito mínimo (Douglas G. Altman and Bland 2007).

  • Perda de participantes no estudo por dados perdidos pode reduzir o poder estatístico (erro tipo II) (Douglas G. Altman and Bland 2007).


Que estratégias podem ser utilizadas quando há expectativa ou ocorrência de dados perdidos?

  • Não existe solução globalmente satisfatória para o problema de dados perdidos (Douglas G. Altman and Bland 2007).

  • Na expectativa de ocorrência de dados perdidos, recomenda-se ampliar o tamanho da amostra com um % correspondente à tal estimativa - embora ainda não corriga potenciais vieses pela perda (Douglas G. Altman and Bland 2007).

  • Na ocorrência de dados perdidos, a análise mais comum compreende apenas os ‘casos completos’, com exclusào de participantes com algum dado perdido nas variáveis do estudo. Em casos de grande quantidade de dados perdidos, pode-se perder muito poder estatístico (erro tipo II elevado) (Douglas G. Altman and Bland 2007).

  • Na ocorrência de dados perdidos, a imputação de dados (substituição por dados simulados plausíveis preditos pelos dados presentes) pode ser uma alternativa para manter o erro tipo II estipulado no plano de análise (Douglas G. Altman and Bland 2007).

Parâmetros


O que são parâmetros?

  • Parâmetros são informações que definem um modelo teórico, como propriedades de uma coleção de indivíduos (D. G. Altman and Bland 1999).


Quais são os parâmetros mais comuns?

  • Parâmetros de tendência central: médial, mediana, moda (Kanji 2006).

  • Parâmetros de dispersão: variância, desvio-padrão, amplitude, intervalo interquartil (Kanji 2006).

  • Parâmetos de proporção: frequência absoluta, frequência relativa (Kanji 2006).

Delineamento de estudos


Pareamento


O que é pareamento?

  • Pareamento significa que para cada participante de um grupo (por exemplo, com alguma condição clínica), existe um (ou mais) participantes (por exemplo, grupo controle) que possui características iguais ou similares relativas a algumas variáveis de interesse (Bland and Altman 1994).

  • As variáveis escolhidas para pareamento devem ter relação com as variáveis de desfecho, mas não são de interesse elas mesmas (Bland and Altman 1994).

  • O ajuste por pareamento deve ser incluído nas análises estatísticas mesmo que as variáveis de pareamento não sejam consideradas prognósticas ou confundidores na amostra estudada (Bland and Altman 1994).

  • A ausência de evidência estatística de diferença entre grupos não é considerada pareamento (Bland and Altman 1994).

Ajuste de covariáveis


Quais os benefícios do ajuste de covariáveis?

  • Incluir a variável de desfecho medida na linha de base como covariável - independentemente da análise ser realizada com a medida pós-tratamento da mesma variável ou a diferença para a linha de base - pode aumentar o poder estatístico do estudo (Kahan et al. 2014).

  • Incluir outras variáveis medidas na linha de base, com potencial para serem desbalanceadas entre grupos após a randomização, diminui a chance de afetar as estimativas de efeito dos tratamentos (Kahan et al. 2014).


Quais os riscos do ajuste de covariáveis?

  • Incluir covariáveis que não são prognósticas do desfecho pode reduzir o poder estatístico do estudo (Kahan et al. 2014).

  • Incluir covariáveis com dados perdidos pode reduzir o tamanho amostral e consequentemente o poder estatístico do estudo (análise de casos completos) ou levar a desvios do plano de análise por exclusão de covariáveis prognósticas (Kahan et al. 2014).


Como lidar com os dados perdidos em covariáveis?

  • Imputação de dados perdidos de uma variável pela média dos dados observada permite estimativas não enviesadas dos efeitos do tratamento, preserva o erro tipo I e aumenta o poder estatístico comparado à análise de dados completos (Kahan et al. 2014).
